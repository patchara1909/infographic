4. การเชื่อมโยงงานกับวัตถุประสงค์โครงงานและภาพรวมกลุ่ม

ถ้า QR Code ถูกพัฒนาและทดสอบแล้ว จะช่วยให้ระบบ 2FA ทำงานได้สมบูรณ์ โดย QR Code เปรียบเสมือน รหัสนักศึกษาดิจิทัล ที่ใช้ก่อนการยืนยันตัวตนด้วย AI ตรวจจับใบหน้า เป็นตัวกลางการเชื่อมต่อระหว่างผู้ใช้กับ Database และ AI

ข้อควรระวัง: หากระบบ QR Code มีข้อผิดพลาด เช่น สแกนไม่ได้ หรือข้อมูล Database ไม่ถูกต้อง จะส่งผลให้ระบบเช็คชื่อโดยรวมทำงานไม่สมบูรณ์
5. การวิเคราะห์ปัญหาและการเรียนรู้

ปัญหาที่พบ: มีปัญหาหลายด้านเกี่ยวกับ QR Code และการเชื่อมต่อ Database ทำให้ต้องใช้เวลาในการแก้ไขเพิ่มขึ้น

  • QR Code ไม่สามารถยืนยันตัวตนคนที่สแกน
  • QR Code ที่เจนบางครั้งสแกนไม่ได้
  • ดึงข้อมูลรูปภาพมาเปรียบเทียบกับบุคคลไม่ได้
  • เรียกกล้องมาสแกน QR ไม่สำเร็จ
  • ดึงข้อมูลจาก Database มาใช้ร่วมกันไม่ได้

แก้ไขด้วย: ค้นหางานวิจัยเพิ่มเติมเกี่ยวกับ QR Code และ Database จนแก้ปัญหาได้ครบ ✅

6. แผนงานระยะถัดไป

ตอนนี้ระบบ QR Code สำเร็จแล้ว แต่ขั้นต่อไปต้องรวมโปรเจคกับเพื่อน ซึ่งอาจมีปัญหาในการเชื่อมต่อ API และ Server

  • นำโปรเจครวมกับเพื่อนในกลุ่ม
  • นำขึ้น Server เพื่อใช้งานจริง (Production)
  • ทดสอบระบบรวม QR + Face AI + Database
